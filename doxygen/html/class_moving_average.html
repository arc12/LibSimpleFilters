Detailed Description

A finite length moving average without weighting.

This filter calculates the average of a set number of previous values. This has a very slow-moving output if a long length is specified. To use: create an instance of the filter and submit new readings using update(). Readings should be sampled at regular (i.e. equal) time intervals.
