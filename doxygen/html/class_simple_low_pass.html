Detailed Description

An infinite length moving average with exponential weighting.

This filter calculates its output based on the previous output and the update value. output[i] = alpha*input[i] + (1-alpha)*output[i-1], where alpha is the smoothing factor and i is the sample/frame (i-1 = previous). Small values of alpha (<<0.5) make for slow-moving output. The time constant, "RC" = sample period * ((1-alpha)/alpha) RC = time for a step change to reach 63.2% of asymptotic value. Cutoff frequency = 1/(2*pi*RC), hence f_sampling/f_cutoff = 2*pi*(1-alpha)/alpha So a "cutoff frequency" of 1/10th the sample rate requires alpha=0.385
This is a an infinite-impulse-response (IIR) single-pole lowpass filter, see http://en.wikipedia.org/wiki/Low-pass_filter and also http://helpful.knobs-dials.com/index.php/Low-pass_filter for more introductory comments.
To use: create an instance of the filter and submit new readings using update().
